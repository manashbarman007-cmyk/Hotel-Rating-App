# Config Server

The **Spring Cloud Config Server** provides centralized configuration management for all microservices. It fetches configuration properties from a Git repository and makes them available to clients (like the API Gateway, User Service, Hotel Service, etc.).

---

## ⚙️ Configuration

### Server
```yaml
server:
  port: 8085
```
