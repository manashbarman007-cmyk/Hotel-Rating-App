# Eureka Server

The **Eureka Server** provides service discovery for the microservices architecture. It allows services like the API Gateway, Config Server, User Service, Hotel Service, and Rating Service to register themselves and discover each other dynamically.

---

## ⚙️ Configuration

### Spring Application
```yaml
server:
  port: 8761
```
